PROGRAMMING LANGUAGES

Terms Definitions
Machine Language First generation language sequences of instructions encoded as numeric digits.
Debugging Locating and correcting errors of a program.
Assembler Program that was developed to convert mnemoic expressions into machine language instructions.
Assembly Language Mnemonic system for representing programs for second generation languages.
Translator A program written to translate programs expressed in high-level primitive language into machine language.
Compiler A translation program that compiles machine instructions into short sequences to simulate activity from single high-level primitive.
Interpreter
  • Implementing third-generation languages
  • Executes the instructions as translated instead of recording the translated versioin for use in the future.
  • Executes a program from its high level form.
Cross-Platform Software Independent of the operating system design and hardware design, it is executable throughout the network.
Natural Language English, German and Latin. Defined by grammar.
Formal Language Programming languages that evolved over time with out formal grammatical analysis.
Machine Independence Does not rely on the characterisics of another machine.
High-Level Programming Language Provides more abstraction for the programmer in that it makes it easier to write and read programs.
Object-Oriented Programming (OOP) Software system viewed as collection of units "objects" that interact to solve the problem.
Scripting Languages Used to perform administrative tasks rather than develop complex programs.
Data Types Encompass the manner in which the data item is encoded and operations that can be performed on the data.
Integer Numeric data consisting of whole numbers. Operations performed on this data: traditional arithmetic and comparisons of relative size.
Float "Real" numeric data that might contain values other than whole numbers.
String Sequence of characters that represent text.
Primitive Data Type Data types that are included as primitives in programming language.
Data Structure Conceptual shape or array of data.
Python Collection Data Structures
List Used to store multiple items in a single variable. One of four built in data types in Python.
Tuple A collection that is ordered and inchangeable.
Set A collection that is: unordered, unchangeable, and inindexed.
Dictionary It is used to store values in key value pairs; orderd, changeable, duplication is not allowed.

Function Set of instructions for performing a task that us used as abstract tool by other program units.
Local Variable A variable declared with a function can only be refrenced within that function.
Global Variable A variable that is non-restricted to a particular part of a program.
Source Program A program in its original form before translation.
Object Program The translated version of a program.
